# Doculess / Docu-less

A web application that helps organizations and businesses manage their documents efficiently.

In addition, this project is also known as the digital transformation to a paperless workflow.

## Detail information

Actually, this web application is the "specialized project" in my university.

I built it with all the technologies/frameworks I listed below, and I also implemented GitHub Actions to make a CI/CD pipeline for both front-end and back-end.

## Tech Stack

**Front-end:** [(link to repository)](https://github.com/bhkhanh/doculess-frontend)

- TypeScript
- Next.js framework
- Tailwind CSS

**Back-end:** [(link to repository)](https://github.com/bhkhanh/doculess-backend)

- Django / Django REST framework
- PostgreSQL database
